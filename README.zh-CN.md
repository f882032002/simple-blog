## 特性

- :bulb: **TypeScript**: 應用程序級 JavaScript 的語言
- :scroll: **區塊**: 通過區塊模板快速建構頁面
- :gem: **優雅美觀**：基於 Ant Design 體系精心設計
- :triangular_ruler: **常見設計模式**：提煉自鐘後台應用的典型頁面和場景
- :rocket: **最新技術線**：使用 React/umi/dva/antd 等前端前沿技術開發
- :iphone: **響應式**：針對不同螢幕大小設計
- :art: **主題**：可配置的主題滿足多樣化的品牌訴求
- :globe_with_meridians: **國際化**：內建業界通用的國際化方案
- :gear: **最佳實踐**：良好的工程實踐祝您持續產出高品質代碼
- :1234: **Mock 數據**：實用的本地數據調試方案
- :white_check_mark: **UI 測試**：自動化測試保障前端產品品質

```
- Dashboard
  - 分析頁

  - 高級表單頁

  - 查詢表格
  - 標準列表
  - 卡片列表
  - 搜索列表（項目/應用/文章）
- 詳情頁
  - 基礎詳情頁
  - 高級詳情頁
- 用戶
  - 用戶中心頁
  - 用戶設置頁

  - 成功頁
```

## 使用

```bash
$ yarn create umi  # or npm create umi

# Choose ant-design-pro:
 Select the boilerplate type (Use arrow keys)
❯ ant-design-pro  - Create project with an layout-only ant-design-pro boilerplate, use together with umi block.
  app             - Create project with a simple boilerplate, support typescript.
  block           - Create a umi block.
  library         - Create a library with umi.
  plugin          - Create a umi plugin.

$ npm install
$ npm start         # visit http://localhost:8000
```

更多信息請參考 [使用文檔](http://pro.ant.design/docs/getting-started)。
